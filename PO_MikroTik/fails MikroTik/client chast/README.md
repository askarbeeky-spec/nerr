# 🌐 SKY Internet - Captive Portal (Клиентский фронтенд)

## 🎯 Назначение

Это Captive Portal (Landing Page) для конечных пользователей SKY Internet, реализующий технологию **Walled Garden** с идентификацией по MAC-адресу.

## ✨ Основные возможности

### 🔒 Walled Garden

- Пользователь подключается к Wi-Fi **SKY_Internet**
- MikroTik автоматически перенаправляет на эту страницу
- Доступ в интернет закрыт, кроме "белого списка":
  - Платежные шлюзы (Mbank, Visa, O!Dengi)
  - Этот Captive Portal

### 👤 Идентификация по MAC

- **Нет паролей** - идентификация только по MAC-адресу устройства
- **Нет регистрации** - моментальный доступ после оплаты
- **Автоматическое распознавание** - при повторном подключении система узнает устройство

### 💳 Три тарифа

1. **1 Час** - 50 сом
   - Быстрый доступ
   - Безлимитный трафик
   - Скорость до 50 Мбит/с

2. **Сутки** - 150 сом (популярный) ⭐
   - Скорость до 100 Мбит/с
   - Приоритет в сети
   - Экономия 70%

3. **Неделя** - 500 сом
   - Максимальный приоритет
   - QoS оптимизация
   - Скорость до 100 Мбит/с

### 🔄 Сценарии использования

#### Новый пользователь:
1. Подключается к Wi-Fi
2. Перенаправляется на Captive Portal
3. Видит приветствие и свой MAC-адрес
4. Выбирает тариф
5. Оплачивает (карта/MBank/O!Dengi)
6. Доступ открывается автоматически

#### Возвращающийся пользователь:
1. Подключается к Wi-Fi
2. Система распознает MAC-адрес
3. Показывает **"С возвращением!"**
4. Отображает остаток времени
5. Предлагает продлить доступ

## 🌊 User Flow

### Шаг 1: Подключение
```
Пользователь → QR-код/открытый Wi-Fi → SKY_Internet
```

### Шаг 2: Переадресация
```
MikroTik → Captive Portal (этот фронтенд)
Internet ❌ | Платежные шлюзы ✅
```

### Шаг 3: Выбор и оплата
```
Выбор тарифа → Способ оплаты → Платеж
```

### Шаг 4: Активация
```
Система привязывает MAC → Открывает доступ → Перенаправление на сайт
```

### Шаг 5: Возвращение
```
Повторное подключение → "С возвращением!" → Показ остатка времени
```

## 🔌 API Интеграция

Фронтенд взаимодействует с бекендом через следующие эндпоинты:

### Проверка сессии
```javascript
POST /api/check-session
{
  "mac_address": "AA:BB:CC:DD:EE:FF",
  "router_id": "router_001"
}
```

### Обработка платежа
```javascript
POST /api/process-payment
{
  "mac_address": "AA:BB:CC:DD:EE:FF",
  "router_id": "router_001",
  "tariff_type": "hour",
  "tariff_name": "1 Час",
  "price": 50,
  "duration_minutes": 60,
  "payment_method": "card"
}
```

### Уведомление партнера
```javascript
POST /api/partner/session-created
{
  "router_id": "router_001",
  "session_id": "sess_xxx",
  ...
}
```

## 🚧 Что дальше

Когда бекенд будет готов:

1. Установите правильный `API_URL` в `script.js`
2. Настройте MikroTik Walled Garden:
   - Добавьте домен Captive Portal
   - Добавьте платежные шлюзы
3. Настройте переадресацию на Captive Portal
4. Отключите `DEBUG` mode (`DEBUG: false`)
