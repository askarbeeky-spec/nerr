🔗 Endpoints

POST /create_and_confirm

Создать платеж с автоподтверждением

GET /payments

Получить список всех платежей

GET /payment/{id}

Получить информацию о конкретном платеже

Подтвердить платеж вручную

🔧 Использование

Через эту страницу: Заполните форму и нажмите "Создать платеж"

Через Python:

import requests

url = "https://eschatological-unpoisonously-cleora.ngrok-free.dev/create_and_confirm"

data = {
    "sender_name": "Иван",
    "sender_account": "KZ123",
    "receiver_name": "Петр",
    "receiver_account": "KZ987",
    "amount": 10000,
    "currency": "KZT",
    "auto_confirm": True
}

response = requests.post(url, json=data)
print(response.json())

Через curl:

curl -X POST https://eschatological-unpoisonously-cleora.ngrok-free.dev/create_and_confirm \
  -H "Content-Type: application/json" \
  -d '{"sender_name":"Иван","sender_account":"KZ123","receiver_name":"Петр","receiver_account":"KZ987","amount":10000,"currency":"KZT","auto_confirm":true}'

⚡ Особенности

  • Автоматическое подтверждение через 1 секунду
  • Платежи хранятся в памяти сервера
  • Доступ через интернет (ngrok)
  • Нет аутентификации (для тестирования)
